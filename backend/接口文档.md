# 后端接口文档

## 基础信息

默认服务地址：

```text
http://127.0.0.1:8080
```

如果启动时指定了备用端口，例如 `--server.port=18083`，请将文档中的 `8080` 替换为实际端口。

## 统一返回格式

后端接口统一返回 `ApiResponse<T>` 格式：

```json
{
  "code": 0,
  "message": "success",
  "data": {}
}
```

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `code` | `number` | 业务状态码，`0` 表示成功，非 `0` 表示失败。 |
| `message` | `string` | 请求处理结果说明。 |
| `data` | `object` / `null` | 实际响应数据，不同接口的数据结构不同。 |

## 1. 健康检查接口

用于验证后端服务是否正常运行。

### 请求信息

```http
GET /api/health
```

完整地址：

```text
http://127.0.0.1:8080/api/health
```

请求参数：无

### 成功响应

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "status": "ok"
  }
}
```

### 响应字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `data.status` | `string` | 后端服务状态，`ok` 表示服务正常。 |

### 验证命令

```powershell
Invoke-RestMethod 'http://127.0.0.1:8080/api/health' | ConvertTo-Json -Depth 8
```

## 2. 数据库连通性检查接口

用于验证后端是否可以正常连接 MySQL，并返回数据库当前时间。

### 请求信息

```http
GET /api/db/ping
```

完整地址：

```text
http://127.0.0.1:8080/api/db/ping
```

请求参数：无

### 成功响应

```json
{
  "code": 0,
  "message": "success",
  "data": {
    "timestamp": "2026-07-20 09:23:43",
    "status": "ok"
  }
}
```

### 成功响应字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `data.timestamp` | `string` | MySQL 当前时间，格式为 `yyyy-MM-dd HH:mm:ss`。 |
| `data.status` | `string` | 数据库连通状态，`ok` 表示连接正常。 |

### 失败响应

```json
{
  "code": 500,
  "message": "Database ping failed: connection timeout",
  "data": {
    "timestamp": null,
    "status": "error"
  }
}
```

### 失败响应字段

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `code` | `number` | 失败时当前返回 `500`。 |
| `message` | `string` | 数据库连接失败原因。 |
| `data.timestamp` | `null` | 连接失败时没有数据库时间。 |
| `data.status` | `string` | `error` 表示数据库连通性检查失败。 |

### 验证命令

```powershell
Invoke-RestMethod 'http://127.0.0.1:8080/api/db/ping' | ConvertTo-Json -Depth 8
```

## 启动与验证

### 默认端口启动

```powershell
cd G:\AI-law-creater\backend
mvn spring-boot:run "-Dspring-boot.run.profiles=local"
```

另开一个 PowerShell 窗口验证：

```powershell
Invoke-RestMethod 'http://127.0.0.1:8080/api/health' | ConvertTo-Json -Depth 8
Invoke-RestMethod 'http://127.0.0.1:8080/api/db/ping' | ConvertTo-Json -Depth 8
```

### 备用端口启动

```powershell
cd G:\AI-law-creater\backend
mvn spring-boot:run "-Dspring-boot.run.profiles=local" "-Dspring-boot.run.arguments=--server.port=18083"
```

备用端口验证：

```powershell
Invoke-RestMethod 'http://127.0.0.1:18083/api/health' | ConvertTo-Json -Depth 8
Invoke-RestMethod 'http://127.0.0.1:18083/api/db/ping' | ConvertTo-Json -Depth 8
```
