检查劳动合同条款中的常见风险

仅识别常见明显风险，不替代完整的律师合同审查。

审查结果

竞业限制补偿约定不明确

warning 高风险

原始片段

“乙方离职后两年内不得从事同类业务，甲方视情况给予一定补偿。”

风险类型

约定不明 / 显失公平

原因

未明确具体补偿金额和发放方式。

修改建议

建议补充明确的月度补偿标准（通常不低于离职前十二个月平均工资的30%）。

法律依据

《最高人民法院关于审理劳动争议案件适用法律问题的解释（一）》第三十六条

工作地点表述宽泛

info 低风险

原始片段

“工作地点：全国范围内根据公司业务需要调配。”

修改建议

虽属常见，但建议尽可能细化核心工作常驻地，避免未来跨省调岗引发争议。
