入职三个月未签订书面劳动合同，现在公司要辞退我，我能要求双倍工资赔偿吗？

gavel

权威结论摘要

可以要求。根据《劳动合同法》规定，用人单位自用工之日起超过一个月不满一年未与劳动者订立书面劳动合同的，应当向劳动者每月支付二倍的工资 [1]。同时，若公司单方面违法辞退，您还可以要求支付违法解除劳动合同的赔偿金（即经济补偿金的二倍）[2]。

建议处理流程

第一步：固定证据（核心关键）

在与公司交涉前，务必先收集能证明存在劳动关系的证据，以及实际工资标准的证据。切勿冲动先办离职手续。

第二步：协商或投诉

携带证据与人事部门沟通，或前往当地劳动监察大队进行投诉调解。
